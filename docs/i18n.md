## Translator

A simple class: support move but forbid copy operation;  
- Use the `FNV-1a` hash algorithm compute the string key and store the hash value as key. Should compute during compile.
- - [Non-crypto hashes in C++: FNV 1/1a]( https://asecuritysite.com/encryption/smh_fnv),

SimpleIME  
- `TranslatorHolder`:  
	 - Be designed `only` the first caller grant permissions to modify `Translator`.  
- Default translate text:  
	 - May keep the key as default text: User should ensure mod files are complete.  
 - Use `.toml` as the translation file format:  
	  - Key should be defined by role, behaviour, or feature.  
	  - Key should inherit the parent key, the full qualified prefix, like:  
		- `Settings.Sidebar.Appearance`
	 - `toml` suppport nested section which help define the key.
	 ```toml
	 [Settings]
	 [Settings.Sidebar]
	 Appearance = "Appearance"
	 ```
